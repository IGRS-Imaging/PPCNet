PPCNet — Tracking Videos
==================================

The videos demonstrating
the clinical applicability of PPCNet for surgical navigation
and reconstruction quality.

Videos
------
1. tracking_stl.mp4
   Real-time phantom-based navigation tracking on the
   CT-derived STL mesh using an optical tracker.
   (1920x1080, 30 fps, 31s)

2. tracking_pointcloud.mp4
   Real-time phantom-based navigation tracking on the
   PPCNet-predicted point cloud using an optical tracker.
   (1920x1080, 30 fps, 31s)

3. prediction_360_rotation.mp4
   360-degree rotation of the PPCNet-predicted 8,192-point
   cloud for a test patient, showing vertebral surface
   coverage and inter-vertebral gap preservation.
   (1918x1014, 30 fps, 43s)

4. gt_vs_prediction.mp4
   Side-by-side synchronized rotation of the ground-truth
   point cloud (green, 5,120 points) and PPCNet-predicted
   point cloud (blue, 8,192 points) for the same patient.
   (1918x1008, 30 fps, 53s)
